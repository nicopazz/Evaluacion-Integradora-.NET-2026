#  Evaluación Integradora .NET 2026

## 📌 Descripción

Este proyecto corresponde a una **evaluación integradora de Desarrollo .NET 2026**, donde se implementa un sistema de gestión para una clínica veterinaria.

El objetivo principal es aplicar conceptos de:

* Programación Orientada a Objetos (POO)
* LINQ
* Records
* Estructuras repetitivas
* Recursividad

---

## 🏥 Contexto

La clínica veterinaria **"Huella Feliz"** necesita digitalizar su sistema, ya que actualmente gestiona todo en papel, generando ineficiencias.

Este sistema permite:

* Registrar pacientes (animales)
* Gestionar consultas veterinarias
* Calcular costos
* Generar reportes y estadísticas

---

## 🚀 Tecnologías utilizadas

* C#
* .NET 10
* LINQ
* Programación Orientada a Objetos

## 🧩 Problema 1 — Sistema de Gestión

Se desarrolla un sistema basado en POO que incluye:

### 🔹 Modelo de clases

* Clase abstracta `Animal`
* Clases derivadas `Perro` y `Gato`
* Interfaz `IVacunable`
* Record `Consulta`

### 🔹 Funcionalidades

* Carga de registros de animales
* Registro de consultas
* Eliminación y alta de datos
* Uso de polimorfismo
* Implementación de interfaces

### 🔹 LINQ

Consultas implementadas:

* Ordenamiento por edad
* Filtro por veterinario y fecha
* Total gastado por paciente
* Estadísticas generales

---

## 🔁 Problema 2 — Lógica avanzada

Extensión del sistema incorporando:

### 🔹 Estructuras repetitivas

* `for` → historial de consultas
* `while` → cálculo de costos
* `do-while` → reportes acumulados

### 🔹 Recursividad

* Búsqueda de animales por ID sin usar bucles

### 🔹 Estructuras de datos

* Arrays → costos por paciente
* Matrices → relación paciente × veterinario

---

## 📊 Funcionalidades destacadas

* ✔ Cálculo de costos por tipo de animal
